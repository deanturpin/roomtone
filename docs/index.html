ROOMTONE

Real-time frequency analyser with musical key detection and bass tone generation

Every room has its own acoustic signature. ROOMTONE discovers those frequencies and uses them to generate complementary bass tones, creating ambient music perfectly tuned to your space. Features live spectrum analysis, MIDI keyboard support, and musical key detection.

Key Features

  • Real-time FFT frequency analysis with musical note identification
  • MIDI keyboard input for live performance
  • Bass tone generation for room tone foundation
  • Musical key detection using harmonic scoring
  • Canvas-based spectrum and waveform visualisation
  • Privacy-first: all processing happens locally
